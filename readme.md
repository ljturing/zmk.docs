# 文档网页

该网页是用 [Docusaurus 2](https://v2.docusaurus.io/) 构建，一个现代静态网站生成工具。

## 许可

ZMK 文档许可 [CC-BY-NC-SA](http://creativecommons.org/licenses/by-nc-sa/4.0/)。

### 安装

```sh
$ npm ci
```

### 本地开发

```sh
$ npm start
```

这条命令会启动一个本地开发服务并打开一个浏览器窗口，大多变更都可以不重启服务实施更新。

### 编译

```sh
$ npm build
```

这条命令会在 `build` 目录里生成静态内容，可以使用任何静态内容托管查看。
